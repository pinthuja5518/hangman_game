PROJECT TITLE 
Hangman Game Using Python

OBJECTIVE 
To develop a simple word-guessing game that improves logical thinking and demonstrates the use of Python programming concepts such as loops, conditions, strings, and lists.

DESCRIPTION 
The Hangman Game is a console-based Python application where the player tries to guess a hidden word one letter at a time. The player has a limited number of attempts. Correct guesses reveal letters in the word, while incorrect guesses reduce the remaining chances. The game ends when the player guesses the word or runs out of attempts.

FEATURES 
✓Random word selection
✓Letter-by-letter guessing
✓Limited number of attempts
✓Displays correctly guessed letters
✓Tracks incorrect guesses
✓Win/Loss result display
✓User-friendly console interface

CONCEPTS USED
✓Variables
✓Data Types
✓Lists
✓Strings
✓Loops (while, for)
✓Conditional Statements (if, else)
✓Functions
✓User Input (input())
✓Random Module (random)

CONCLUSION 
This project demonstrates basic Python programming skills and helps beginners understand game development logic, user interaction, and problem-solving techniques.
